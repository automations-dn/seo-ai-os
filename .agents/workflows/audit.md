---
description: Run a full SEO and GEO audit for a website.
---

1. Start by identifying the target URL.
2. Use the `seo-audit` skill to perform a comprehensive Technical, Content, and Schema audit.
3. Use the `geo-audit` skill to analyze AI citability, platform presence, and LLM visibility.
4. If DataForSEO credentials are provided in the environment, use the `seo-dataforseo` skill for live SERP and backlink data.
5. Synthesize all findings into a final `FULL-AUDIT-REPORT.md` and an `ACTION-PLAN.md`.
6. Provide a scoring summary from both SEO (0-100) and GEO (0-100) perspectives.
